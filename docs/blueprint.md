# **App Name**: GridGuard 2030

## Core Features:

- Real-time Grid Monitoring Dashboard: A comprehensive dashboard providing live display of national AT&C losses, energy flow, and active alerts, using real-time database feeds.
- AI Theft Pattern Detection: Utilize an AI-powered tool to identify and classify electricity theft patterns in real-time, displaying confidence scores and evidence reports.
- Predictive Asset Maintenance: Employ AI to predict transformer failures and health anomalies based on live sensor data, facilitating proactive maintenance and work order generation.
- Interactive National Grid Map: A layered, interactive map visualizing state-wise AT&C losses, substation health statuses, major transmission lines, and live alert locations.
- Secure Blockchain Billing Ledger: A blockchain-based system for immutable recording and transparent verification of energy transactions, enhancing billing accuracy and audit trails.
- Custom Analytics Report Generation: Generate on-demand, customizable reports and analytics on grid performance, theft cases, and asset health across different regions and timeframes.

## Style Guidelines:

- Primary interactive color: Electric blue (#00D4FF), serving as the main accent for active states, interactive elements, and critical information against a dark theme.
- Base color scheme: Predominantly deepest navy black (#050A14) for the background, complemented by layered darker navy tones (#0A1628, #0D1F3C) to establish a dark, industrial aesthetic.
- Dynamic accent colors: Neon green (#00FF88) for live data streams and active indicators; Amber (#FFB800) for warnings and alerts; Red (#FF4444) for critical warnings, ensuring immediate visual cues.
- Display font: 'Exo 2' (sans-serif) for headlines and titles, offering a geometric and technical feel. Note: currently only Google Fonts are supported.
- Body font: 'IBM Plex Sans' (sans-serif) for body text and descriptive content, ensuring high readability and a professional appearance. Note: currently only Google Fonts are supported.
- Monospace font: 'JetBrains Mono' (monospace sans-serif) for live numbers, data streams, and code, providing a distinct terminal-like look. Note: currently only Google Fonts are supported.
- Use a consistent set of minimalist, line-based icons (e.g., from Lucide React), adopting the electric blue/neon green palette for status indicators and interactive elements, fitting the futuristic theme.
- Structured grid-based layouts with a fixed top navigation, a collapsible left sidebar, and various multi-column arrangements to create a dashboard-centric 'mission control' experience.
- Dynamic and responsive elements including full-screen interactive maps with floating control panels and adaptable split-screen designs, optimized for various device sizes.
- Dynamic micro-interactions using Framer Motion for fluid page transitions, scroll-triggered element animations, and expressive data update feedback (e.g., flashing KPI backgrounds) to enhance interactivity.
- Immersive visual effects such as a Three.js animated particle network on the landing page, subtle CRT-like scanline overlays, and character-by-character live data stream animations to amplify the industrial-futuristic theme.
